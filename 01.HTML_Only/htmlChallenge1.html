Lists can be ordered (<ol>) or unordered (<ul>), and each item is marked with <li>.

Images

The <img> tag is used to embed images. The src attribute contains the image URL, and alt provides alternative text.

Links

Links are created with the <a> tag. The href attribute contains the URL of the linked page or resource.
